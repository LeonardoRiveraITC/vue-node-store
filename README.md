## Project setup
1.Clonar el repositorio 
```
https://github.com/LeonardoRiveraITC/vue-template
```
2. Instalar paquetes
```
npm install
```
3. Levantar servidor de desarrollo
```
npm run dev
```

### Compiles and hot-reloads for development
